B) HTML vs PHP - top

I've included both a fully HTML version of this template as well as a PHP version, located in the folders _html and _php. You can use either one of these, although I highly suggest using the PHP version as it will make everything much easier on you. The HTML version is included solely for users whose servers do not support PHP.

The PHP version has an includes folder that contains the header, footer, and sidebar of the site. This way, you only need to make edits to those areas once rather than on every single page as you have to do with the HTML version of the template.

 

C) General Markup Info & Structure - top

The PHP version of Campaign uses basic PHP functionality. This should make things easier for you because in the includes folder there is a header.php, footer.php, and sidebar.php. These files are pulled on to every page so you only need to make changes that show up on every page once.

The HTML version users will need to edit each file individually. The following applies to both versions of this template.

There are a few classes and stylesheets that you need to know of in order to customize Campaign HTML.

  1. Fonts

    • In includes/header.php (or in the header of each file if you're using the fully HTML version of this template), line 16 you should see 

      <link rel="stylesheet" type="text/css" href="fonts/style-nevis.css" media="screen" />

      This is the line that sets your heading font. You can change style-bitter.css to any of the following: bitter, droidsans, droidserif, franchise, museo, nevis, or rokkitt.

  2. Body Classes

    On line 23 of includes/header.php (or in the header of each file if you're using the fully HTML version of this template), you'll see 

    <body class="content_left bg_linen body_boxed scheme_blue">

    Each of those classes can be changed to change the style of the site. I'll break them down:

    • The first class can be either content_left or content_right. These set which side of the screen the content will be on. The sidebar will be on the opposite.
    • The second class sets the background pattern. This only works with the body_boxed class (which is the next class that's set). The options for the background patterns are bg_linen, bg_freckles, bg_cork, bg_fabric, bg_pinstripes, bg_none.
    • The third class sets the body style, either body_boxed which is a 'boxed in style' with a ribbon for the menu, or the body_span which is a full body width layout. Note: if you want to use the sticky header (which we'll get to later), it will only work with the body_span style.
    • The last sets the colors for the color scheme in the header. The options are scheme_blue, scheme_red, scheme_green, scheme_yellow.

D) The Footer - top

The footer, located in includes/footer.php (or in the bottom of each file if you're using the fully HTML version of this template) has a slogan area that you can place your slogan into. Below that there are 4 'widget' areas. I've included a few different styles.

Below the widgets are the social network icons. Go ahead and put the links to your accounts in there, or remove the ones you're not using.

On the bottom is the 'Paid For Disclaimer' box and the copyright info. The disclaimer this is required in the USA by the FEC for political committees. You can learn more about that here.

E) Contact Form - top

Note that the working contact form DOES NOT work in the HTML version, only the the PHP version of this template.

The contact form is on page-contact.php. Open up that file and put your website's name and your email on lines 31 and 32, respectively. Other than that, the only other thing you may need to change is the action="page-contact.php" on line 58. If you change the name of the actual page-contact.php file, you must change it here too, otherwise your site will 404 when somebody fills out the contact form.

F) Extras - top

There are six extra little classes you may want to use in Campaign: Half Width, Quarter Width, Three Quarters Width, an Alert Box, a Warning Box, and a Button. To use them, simply drop the follow code in:

  • <div class="half_wide"> This will be half the page width </div>
  • <div class="quarter_wide"> This will be one fourth the page width </div>
  • <div class="threequarter_wide"> This will be three fourths the page width </div>
  • <div class="alert"> This will be a yellow alert box </div>
  • <div class="warning"> This will be a red warning box </div>
  • <a class="button"> This will create a button </a>

G) Sticky Header - top

This template also has the option of a sticky header. This keeps the header at the top of the site at all time, but it will only work with the body class of body_span, not body_boxed.

To enable the sticky header, you must uncomment two pieces of javascript. The first is the 4th line from the bottom of the footer.

<!--  <script type="text/javascript" src="includes/js/jquery.sticky.js"></script> // uncomment this to use the sticky header -->
The second is in includes/js/campaign.js on lines 33 to 35. 
/* uncomment this to turn on the sticky header. This only works properly with the body class "body_span"
	J("#header").sticky({topSpacing:0});
*/

Remove those comments and presto, instant sticky header.
